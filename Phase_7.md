# Phase 7: Dashboard / UI

## Goal
Build a simple, usable interface for uploading assignments and viewing results.

## Tasks
- Upload page: drag-and-drop or file picker for PDF/DOCX/TXT
- Loading state while processing
- Results page:
  - Overall similarity score (big, clear number/percentage)
  - Highlighted text view showing matched sections
  - List of matched sources
  - Download report button
- (Optional) Teacher dashboard showing all past submissions and scores

## Deliverable
A working frontend connected to the backend API, where a user can upload a file and see the plagiarism report on screen.
