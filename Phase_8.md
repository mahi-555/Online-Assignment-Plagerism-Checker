# Phase 8: Testing & Deployment

## Goal
Test the full system end-to-end and deploy it live.

## Tasks
- Test with known plagiarized and original documents
- Test edge cases: empty files, scanned/image PDFs, very large files
- Fix bugs found in extraction, scoring, and report generation
- Deploy backend (Render/Railway)
- Deploy frontend (Vercel/Netlify)
- Set environment variables (API keys, database URL) on the hosting platform
- Write final documentation and usage instructions in `README.md`

## Deliverable
A live, publicly accessible plagiarism checker web app.
