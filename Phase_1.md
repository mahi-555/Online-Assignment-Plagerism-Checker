# Phase 1: Project Setup

## Goal
Set up the base project structure and tech stack.

## Tasks
- Create GitHub repository
- Set up backend (Python) and frontend (React or plain HTML/CSS/JS) folders
- Install core dependencies:
  - Backend: `fastapi`, `uvicorn` (or `flask`), `python-docx`, `PyPDF2`, `scikit-learn`
  - Frontend: React app via `npx create-react-app` (or plain HTML)
- Set up `.env` file for API keys (search API key, database URL)
- Create initial `README.md`
- Set up database (MongoDB Atlas free tier or PostgreSQL)

## Deliverable
A working empty project skeleton that runs locally (backend server starts, frontend loads a blank page).
