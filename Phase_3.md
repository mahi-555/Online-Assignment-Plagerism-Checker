# Phase 3: Text Preprocessing

## Goal
Clean and normalize extracted text so it can be reliably compared.

## Tasks
- Lowercase all text
- Remove punctuation and special characters
- Remove stopwords (using NLTK or spaCy)
- Tokenize into words and sentences
- Lemmatize words to their root form
- Store both the original text and the cleaned/tokenized version

## Deliverable
A `text_cleaner.py` module that takes raw text and returns cleaned, tokenized text ready for similarity comparison.
